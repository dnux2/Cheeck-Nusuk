## 🔴 CRITICAL FIXES (Must Fix)

### 1. Wrong AI Model Name — server/routes.ts
The model name `gpt-5.1` does not exist in OpenAI. The AI Translator will completely fail.
```
// ❌ Wrong:
model: "gpt-5.1"

// ✅ Fix:
model: "gpt-4o-mini"
```

### 2. Wrong Schema Type — shared/schema.ts
`pilgrimId` in the `emergencies` table is defined as `serial` (auto-increment) instead of `integer`. This is a foreign key and must be `integer`.
```
// ❌ Wrong:
pilgrimId: serial("pilgrim_id").references(() => pilgrims.id)

// ✅ Fix:
pilgrimId: integer("pilgrim_id").references(() => pilgrims.id)
```
Also import `integer` from `drizzle-orm/pg-core`.

### 3. Runtime Crash in resolveEmergency — server/storage.ts
The function does not handle the case where the emergency ID does not exist, causing an unhandled crash.
```
// ✅ Add this null check after the DB query:
if (!resolved) throw new Error(`Emergency with id ${id} not found`);
```

### 4. Add CORS Middleware — server/index.ts
Add basic CORS support to prevent cross-origin request errors.
```
// Add at the top of server/index.ts:
import cors from 'cors';
app.use(cors());
```
Install: `npm install cors @types/cors`

### 5. Remove Unused Fonts — client/index.html
The HTML file loads 20+ Google Fonts which severely slows down the app. Keep only the fonts actually used.
```
// ✅ Keep ONLY these two font imports and remove all others:
// 1. Cairo + Tajawal (Arabic)
// 2. Plus Jakarta Sans (English)
```
